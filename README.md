# My Dotfiles
These is my collection of configuration files for my desktop.
Please note: My yabai configuration is stored inside my .zshrc file

Softwares Used:
* WM: Yabai
* bar: Simple Bar
* editor: NVIM
* Widgets: Ubershict
* Discord: BetterDiscord
* shell: ZSH
* terminal: alacritty
* File Manager: ranger
* Browser: Firefox
